## Install

Via Composer:

~~~ bash
$ composer require kaloa/filesystem
~~~


## Requirements

The following PHP versions are supported:

- PHP 5.3
- PHP 5.4
- PHP 5.5
- PHP 5.6
- PHP 7
- HHVM


## Testing

(Tools are not included in this package.)

~~~ bash
$ phpunit
~~~

Further quality assurance:

~~~ bash
$ phpcs --standard=PSR2 ./src
$ phpmd ./src text codesize,design,naming
~~~
